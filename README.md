# SyncCoordinator Documentation Site

The official documentation site for [SyncCoordinator](https://github.com/clavisflow/SyncCoordinator), a self-hosted platform for synchronizing business data safely across SQL Server, MySQL, and PostgreSQL.

The site provides product overviews, architecture and workflow guides, setup instructions, and synchronized user manuals in English and Japanese.

[View the documentation](https://synco.clavisflow.net/)

## Tech stack

- Next.js 16
- React 19
- TypeScript 5.9
- Tailwind CSS 4
- vinext and Vite
- GitHub Pages
